# CSEC_DATA_SIENCE_BOOTCAMP_TASK_1
Python fundamentals project demonstrating problem-solving, functions, and CLI interaction through a prime checker, Fibonacci generator, and calculator application.


## Level 1: Prime Number Checker
This program checks whether a number is prime using a loop and condition.

## Level 2: Fibonacci Sequence
Generates Fibonacci numbers up to a given number of terms.

## Level 3: CLI Calculator
A command-line calculator that performs basic operations:
- Addition
- Subtraction
- Multiplication
- Division
